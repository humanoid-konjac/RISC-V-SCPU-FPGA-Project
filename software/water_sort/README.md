# Water Sort 可移植 C 核心

该目录包含 FPGA 倒水排序游戏共用的纯 C 核心、终端外壳和主机测试。核心使用固定数组，不依赖操作系统、标准库或动态内存。

当前支持三档难度：

| 难度 | 颜色 | 空管 | 有效试管 |
|---|---:|---:|---:|
| EASY | 4 | 2 | 6 |
| NORMAL | 6 | 1 | 7 |
| HARD | 7 | 1 | 8 |

关卡由 32 位种子确定。`xorshift32` 对经过求解验证的基础布局执行颜色和试管等价置换，因此相同“难度 + 种子”在主机与 FPGA 上逐层一致且保证可解。合法倾倒以 1 字节记录来源、目标和层数，最多保存 2048 步；历史满时拒绝新倾倒，以保证仍可撤回开局。

## 构建与测试

```bash
make
make test
make run
```

测试会对每档检查 10001 个种子，并覆盖确定性、颜色/空管不变量、动态光标、重新生成、倾倒和完整撤销。当前也已通过 AddressSanitizer 与 UndefinedBehaviorSanitizer。

终端外壳使用 `a/d` 移动、`e` 选择/倾倒、`c` 取消、`u` 撤销、`r` 同种子重开、`q` 退出。
