# Water Sort 可移植 C 核心

该目录包含 FPGA 倒水排序游戏共用的纯 C 核心、终端外壳和主机测试。核心使用固定数组，不依赖操作系统、标准库或动态内存。

当前支持三档难度：

| 难度 | 颜色 | 空管 | 有效试管 |
|---|---:|---:|---:|
| EASY | 4 | 2 | 6 |
| NORMAL | 6 | 1 | 7 |
| HARD | 7 | 1 | 8 |

每档包含 12 个结构不同的关卡。电脑端生成器随机打乱颜色层，使用 BFS 验证可解性和最短解长度，按难度筛选后压缩进 `level_catalog.h`。完整算法、阈值和目录格式见 `LEVEL_GENERATION.md`。合法倾倒以 1 字节记录来源、目标和层数，最多保存 2048 步；历史满时拒绝新倾倒，以保证仍可撤回开局。

## 构建与测试

```bash
make
make test
make run
```

测试会先重新生成并逐字节核对目录，再检查全部 36 关的颜色/空管不变量、最短解记录、动态光标、重开、倾倒和完整撤销。当前也已通过 AddressSanitizer 与 UndefinedBehaviorSanitizer。

终端外壳使用 `a/d` 移动、`e` 选择/倾倒、`c` 取消、`u` 撤销、`r` 重开当前关、`q` 退出。
