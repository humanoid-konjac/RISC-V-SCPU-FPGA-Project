# Water Sort 主机端实现

这里是 FPGA 倒水排序游戏的 Step 1：不依赖板级 MMIO 的纯 C 游戏核心、终端交互程序和单元测试。

游戏核心只使用固定大小数组和整数运算，不使用动态内存。数组按从底到顶保存，每种颜色使用 `1`～`6`，`0` 表示空层。这一核心将在后续阶段由主机终端外壳和 FPGA 裸机主循环共同调用。

## 构建与测试

```bash
make
make test
make run
```

交互程序使用 `a`/`d` 左右移动，`e` 选择或倾倒，`x` 取消，`r` 重置，`q` 退出。主机端字符命令只是测试外壳；FPGA 版本会把 PS/2 按键事件映射到同一组核心函数。

## Step 1 完成条件

- 固定的 8 管、6 色、2 空管关卡能够重置和完整求解。
- 非法来源或目标不改变试管状态，也不增加步数。
- 一次倾倒移动来源顶部连续同色层数与目标剩余容量中的较小值。
- 空管允许存在；非空管必须恰好四层同色才算通关。
- 已知 21 步解通过自动测试，通关后仅允许重新开始。
